- 2.工程里面已经有爬取后的rent.csv文件，可以删除，然后执行命令`scrapy crawl fangjia -o rent.csv -t csv`生成csv文件

- 3.爬取完成后，执行命令`python -m SimpleHTTPServer 3000`,然后打开`http://localhost:3000/`,点击打开demo.html，导入上面生成的rent.csv文件即可。
